Privacy stance

Your data stays yours.

We minimize what we keep and how long we keep it. File names are sanitized and saved via Django storage under media/; you can clear sessions anytime.

  • No API keys in code; OpenAI key is pulled from OPENAI_API_KEY.
  • Session chat capped and clearable; avoids unbounded growth.
  • Uploads stored under predictable folders (uploads/, captures/).
Operational notes
  • This is not medical advice. For clinical needs, consult licensed professionals.
  • FER models can inherit dataset bias; please pair with human judgment.
  • Consider configuring automatic cleanup for media if deployed long-term.
